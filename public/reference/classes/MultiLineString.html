    Class MultiLineString

    Multi-line feature class. 多线要素类

    Description

    Represents a multi-line feature in the 3D scene, inheriting from the Line class. Supports display of multiple line segments, each manageable independently.

    表示3D场景中的多线要素，继承自Line类 支持多条线段的组合显示，每条线段可独立管理
