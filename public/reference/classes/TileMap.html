    TileMap Class 瓦片地图类

    Description

    getDataZoom

    • getDataZoom(): number

      获取当前地图的“数据缩放级别”

      Returns number

      Description

      遍历底图图层的瓦片树，返回 showing + inFrustum 瓦片中的最大 tile.z。 注意：这个值最大受底图 TileLayer.minLevel/maxLevel 以及数据源 maxLevel 限制， 比如数据只到 18 级，这里最大也就是 18。
