    • project(lon: number, lat: number): { x: number; y: number }

      地理坐标转投影坐标（墨卡托投影） Convert geographic coordinates to projection coordinates (Mercator projection)

      Parameters

      • lon: number

        经度（单位：度） Longitude (unit: degrees)

      • lat: number

        纬度（单位：度） Latitude (unit: degrees)

      Returns { x: number; y: number }

      投影坐标 {x, y}（单位：米） Projection coordinates {x, y} (unit: meters)

      Description

      实现公式： Implementation formula: x = R * λ y = R * ln[tan(π/4 + φ/2)] 其中R为地球半径，λ为经度弧度，φ为纬度弧度 Where R is the Earth radius, λ is the longitude in radians, and φ is the latitude in radians.

      Overrides Projection.project

    • unProject(x: number, y: number): { lat: number; lon: number }

      投影坐标转地理坐标（反墨卡托投影） Convert projection coordinates to geographic coordinates (Inverse Mercator projection)

      Parameters

      • x: number

        投影坐标X（单位：米） Projection coordinate X (unit: meters)

      • y: number

        投影坐标Y（单位：米） Projection coordinate Y (unit: meters)

      Returns { lat: number; lon: number }

      地理坐标 {lat, lon}（单位：度） Geographic coordinates {lat, lon} (unit: degrees)

      Description

      实现公式： Implementation formula: λ = x / R φ = 2 * arctan(e^(y/R)) - π/2 其中R为地球半径，λ为经度弧度，φ为纬度弧度 Where R is the Earth radius, λ is the longitude in radians, and φ is the latitude in radians.

      Implementation of IProjection.unProject

      Overrides Projection.unProject
