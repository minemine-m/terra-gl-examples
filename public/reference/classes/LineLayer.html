    Class LineLayer

    Line Layer class. 线图层类

    Description

    Specialized layer for managing Line features, inheriting from OverlayLayer base class. Provides functions for adding, removing, and rendering line features. 用于管理线要素的专用图层，继承自OverlayLayer基类。 提供线要素的添加、删除、渲染等功能。
